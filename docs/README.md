# Documentación del sitio Infinite Drive

Solo los documentos necesarios para entender la **arquitectura actual** y las **guías de diseño** están en esta carpeta. El resto se ha movido a **`archive/`** (fases de desarrollo, auditorías puntuales, etc.).

---

## Documentos esenciales

| Documento | Uso |
|-----------|-----|
| **[ARCHITECTURE.md](ARCHITECTURE.md)** | Arquitectura general: contenido, i18n, rutas, vistas, componentes. Punto de entrada para entender cómo está montado el sitio. |
| **[DESIGN-TOKENS-AND-STYLES.md](DESIGN-TOKENS-AND-STYLES.md)** | Tokens de diseño, componentes de UI (props, variantes, cuándo usar cada uno), reglas de estilo y referencia rápida para ediciones. |
| **[EDITING-AND-STRUCTURE.md](EDITING-AND-STRUCTURE.md)** | Dónde editar qué: copy, rutas, tokens, componentes, páginas. Cómo añadir secciones o páginas sin duplicar código. |

Para **despliegue y ejecución local**, ver el **README.md** y **DEPLOY.md** en la raíz del proyecto.
